## Description

`Answers Time` is a game where you can create, edit and play quizes. 
It was made using the `Python` programming language and the `Pygame-CE` library.

This project is a hobby project I started to learn Python - and it ended up being one of my college project too.

### Features:

- Quiz editor
- Take quizes with results
- Various question types
- Windows 7+ compatible
- Completely free
